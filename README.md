## 项目结构
```
|-- api 路由访问的方法
|-- config 配置相关
|-- core 核心配置
|-- docs 文档 swager
|-- global 公用相关
    |-- response 相应值封装
|-- initialize 初始化加载
|-- middleware 中间件
|-- model 对数据的定义与封装
    |-- request 请求相关封装
|-- router 请求路由的封装与转发
|-- service 业务代码
|-- utils 工具
|-- main.go 程序入口
|-- README.md 我自己
```
